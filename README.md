# Ndk_demo
* 作者 AndryYu
* 概述 这是一个使用Android studio进行NDK开发的简单的小应用，主要功能是通过native方法从C层返回一段英文字符串。
* 目的 通过实际操作熟悉在Android studio下NDK开发。如果想详细了解，可以查看我的一篇文章[AndroidStudio下NDK开发流程](http://www.jianshu.com/p/8d22a1075432)
